

# Navigator Object:

```js
var languages=navigator.languages;

      var clearAppBadge=navigator.clearAppBadge;

      var maxTouchPoints=navigator.maxTouchPoints;

      var cookieEnabled=navigator.cookieEnabled;

      var userAgent=navigator.userAgent;

  

      document.write(languages+"<br>");

      document.write(clearAppBadge+"<br>");

      document.write(maxTouchPoints+"<br>");

      document.write(cookieEnabled+"<br>");

      document.write(userAgent+"<br>");
```
